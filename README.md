# Onobe × LAFC Pilot Prototype

Static mobile-first demo of the LAFC Matchday Prediction Game from the pilot proposal.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Screens

| Route | Description |
|-------|-------------|
| `/` | Join — QR scan entry point |
| `/match` | Hub — points, rank, quick actions |
| `/predict` | Four prediction question types |
| `/leaderboard` | Live rankings |
| `/rewards` | Points redemption catalog |

## Build

```bash
npm run build
npm start
```
